# Bibliography

## Methods & software

- **ArchR** — analysis of single-cell chromatin accessibility data.
  <https://www.archrproject.com/>
- **SnapATAC2** — scalable single-cell epigenomics analysis.
  <https://github.com/kaizhang/SnapATAC2>
- **SpatialGlue** — spatial multi-omics integration.
  <https://github.com/JinmiaoChenLab/SpatialGlue>
- **STAGATE** — spatial clustering via graph attention auto-encoders.
  <https://github.com/zhanglabtools/STAGATE>
