# Utils

Platform-agnostic, single-step Workflows used across modalities for common data
operations.

| Workflow | Purpose |
|---|---|
| [downsample reads](downsample.md) | Downsample paired FASTQ reads to a target read count. |
| [ftp downloader](ftp-downloader.md) | Download data from an FTP URL into Latch Data. |
| [merge fastq](merge-fastq.md) | Merge multiple FASTQ files (e.g. across lanes) into one. |
